      ): Promise<PaginatedResponse<T[]>>

      Queries the auto transactions of an account.

      Type Parameters

      • T extends AutoTransactionResponse

        The type of the transaction to be returned.

      Parameters

      • args: {
            accountAddress: AccountAddressInput;
            options?: {
                count?: number;
                block_height?: number;
                cursor?: string;
                ascending?: boolean;
            };
        }
        • accountAddress: AccountAddressInput

          The address of the account to query.

        • Optionaloptions?: { count?: number; block_height?: number; cursor?: string; ascending?: boolean }
          • Optionalcount?: number

            The number of transactions to return.

          • Optionalblock_height?: number

            Starting block height (inclusive). Optional. The block height at which to start lookup for transactions. If provided, returns :count of transactions starting from it in the specified order. For order see :ascending flag.

          • Optionalcursor?: string

            Cursor specifying where to start for pagination. Use the cursor returned by the previous request when making the next request. If provided, returns :count of transactions starting from this cursor in the specified order. For order see :ascending flag. If not specified, the lookup will be done based on the :block_height parameter value.

          • Optionalascending?: boolean

            Whether to return the transactions in ascending order.

      Returns Promise<PaginatedResponse<T[]>>

      A Promise that resolves to an array of AutoTransactionResponse objects representing the auto transactions of the account with optional cursor.

      Note

      If a :cursor is specified then this field will be ignored.

      Note

      If both :cursor and :block_height are specified then :cursor has precedence.

      Note

      Maximum number of items to return default is 20 and maximum is 100.

      Example

      const supra = new SupraClient({ network: Network.TESTNET });

      async function runExample() {
         const accountAddress = "0x1";
         const {response: accountAutoTransactions, cursor} = await supra.account.getAccountAutoTransactions({ accountAddress: accountAddress });
         console.log(accountAutoTransactions);
      }
